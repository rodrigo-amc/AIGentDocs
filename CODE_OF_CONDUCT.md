# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, assume good faith, focus criticism on ideas and code — never on people — and help keep this a welcoming space for contributors of every experience level.

To report unacceptable behavior, contact the maintainer: [@rodrigo-amc](https://github.com/rodrigo-amc).
