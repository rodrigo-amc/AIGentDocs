# @aigenticdocs/standard

The [AIGenticDocs](https://github.com/rodrigo-amc/AIGentDocs) docs-as-code standard, packaged for distribution. Contains `docs/standard/`: agent operating instructions, structure guides, agent profiles, and templates.

This package is a build-time copy — the source of truth lives at [`docs/standard/`](https://github.com/rodrigo-amc/AIGentDocs/tree/main/docs/standard) in the repository. Its version mirrors the standard's `changelog.yaml`.

You probably want the CLI instead: [`aigenticdocs`](https://www.npmjs.com/package/aigenticdocs) (`npx aigenticdocs init`). Prefer no tooling at all? Each release also ships a standalone bundle on [GitHub Releases](https://github.com/rodrigo-amc/AIGentDocs/releases).
