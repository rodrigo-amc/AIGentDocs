# @aigenticdocs/core

Shared domain model and validation logic for the [AIGenticDocs](https://github.com/rodrigo-amc/AIGentDocs) tooling: frontmatter parsing, the deterministic lint rule engine, scaffolding, adapter generation, and standard updates.

You probably want the CLI instead: [`aigenticdocs`](https://www.npmjs.com/package/aigenticdocs). This package exists so the CLI and the future MCP server share one implementation.
